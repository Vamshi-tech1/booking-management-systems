# 📅 Booking Management System
## 📖 Description
A Django CRUD (Create, Read, Update, Delete) web application for managing bookings.
## ✨ Features
- Create a new booking
- View all bookings
- Update existing bookings
- Delete bookings
- Django Admin Panel
## 🛠️ Technologies Used
- Python
- Django
- SQLite3
- HTML
- CSS
## 📁 Project Structure
ProjectK/
│── booking1_app/
│── ProjectK/
│── manage.py
│── db.sqlite3
## 🚀 How to Run
1. Clone the repository
2. Install Django
```bash
pip install django
```
3. Run migrations
```bash
python manage.py migrate
```
4. Start the server
```bash
python manage.py runserver
```
5. Open in your browser
```
http://127.0.0.1:8000/
```
